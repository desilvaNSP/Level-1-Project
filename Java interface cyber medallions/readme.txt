[ENGLISH]

>> SOFTWARE INSTALLATION
  
  - Unpack the demo zip file in a temporary folder.
  - Run setup.exe.
  - Follow setup instructions until installation end.

  - If the setup asks the following question:
      A file being copied is older than the file currently on your system.
      Do you want to keep this file?
      --> Reply 'Yes'

  - In case of setup problems:
    1. Restart your PC and login as Administrator.
    2. Disable antivirus software.
    3. Run the software setup again.

>> SOFTWARE UNINSTALL

  - To uninstall the software from your PC, please use the
    Windows Start > SMSList > Uninstall menu.
  - Removing the software from the Windows Control Panel
    might not perform a complete removal of related components.

[ITALIANO]

>> INSTALLAZIONE DEL SOFTWARE

  - Scompattare il file zip demo in una directory temporanea a piacere.
  - Eseguire setup.exe.
  - Seguire le istruzioni a video fino al termine dell'installazione.

  - Se la procedura di installazione propone la seguente domanda:
      A file being copied is older than the file currently on your system.
      Do you want to keep this file?
      --> Rispondere 'Yes'

  - In caso di problemi con il setup:
    1. Riavviare il PC ed effettuare login coma Amministratore.
    2. Disattivare eventuali antivirus.
    3. Eseguire nuovamente l'installazione.

>> DISINSTALLAZIONE DEL SOFTWARE

  - Per disinstallare il software dal vostro PC, richiamare il menu
    Windows Start > SMSList > Uninstall.
  - La rimozione del software da Pannello di Controllo potrebbe
    non eseguire una rimozione completa dei relativi componenti.
